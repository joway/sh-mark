# SH Mark 

!> 一站式查询 Unix 平台常用操作

## Contribution

1. [提交 issues](https://github.com/joway/sh-mark/issues/new) , 推荐 issues 格式:
	- 解决了什么问题
	- 在什么平台下适用 (精确到版本号)
	- 脚本/命令内容

2. 或直接修改 markdown 文件 , 提交 PR 。注释同上。
